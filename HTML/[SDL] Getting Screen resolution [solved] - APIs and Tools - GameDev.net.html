I was wondering. Is there a way to get the current screen resolution of the system. I'm working in Linux and if the screen resolution is different than what I set my SDL app to (and set it to fullscreen) it crashes. I'd like to be able to find the system screen resolution and change where things are placed on the screen accordingly. Thanks! [Edited by - SigmaX on September 28, 2006 12:28:49 PM]

SDL_GetVideoInfo returns a SDL_VideoInfo structure which has .current_w and .current_h which is the current resolution.
