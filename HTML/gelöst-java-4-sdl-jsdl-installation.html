Hi,
du hast entscheinend nur eine JRE ( "Java Runtime Environment" Version )

Zitat

Perhaps JAVA_HOME does not point to the JDK.
It is currently set to "/usr/lib/jvm/java-1.5.0-gcj-4.2-1.5.0.0/jre"


JAVA_HOME kann somit gar nicht zu einem JDK ( Java Development Kit ) zeigen.
Soviel ich weiß enthält nur das JDK den Javac ( Javacompiler )

Das würde heißen, du schmeißt JRE runter und installierst die neuste Version von dem JDK.
Ich glaube die neuste sollte JDK 1.6 sein, welche du unter der SUN Homepage auffinden kannst, oder in den Repos.

Hi,

Quellcode

 
1

/usr/bin/javac

Das ist ein Link, der wurde erstellt, um nicht in das Verzeichnis von javac gehen zu müssen und ./javac einzugeben.
Folge dem Link und du kommst zu javac. ( ls -al | grep javac )
Ist aber schon komisch, dass du die Verknüpfung von Javac hast, aber das Verzeichnis, von deinem JDK ( wo javac liegt ) nicht angezeigt wird

Wenn das kompilieren finktioniert, dann ist der Java-Kompiler auch vorhanden!
Wo hast du denn JDK (neu) reininstalliert?
Vllt. nochmal "whereis javac" ?
Dann nimmst du den Pfad zur JDK Version und schreibst ihn in die bashrc.
Danach ausloggen und "echo $JAVA_HOME".

Hi,
javac befindet sich auch in

Zitat

/usr/lib/jvm/java-6-sun-1.6.0.03/bin/

Das JAVA_HOME-Verzeichnis ist damit

Zitat

/usr/lib/jvm/java-6-sun-1.6.0.03


Das /usr/bin/javac nach /etc/alternatives/javac zeigt macht nichts, denn das wiederum sollte auf

Zitat

/usr/lib/jvm/java-6-sun-1.6.0.03/bin/javac
zeigen.

Somit gilt:

javac liegt int deinem "/usr/lib/jvm/java-6-sun-1.6.0.03/bin" Verzeichnis !
javac wird nach /etc/alternatives verlinkt
javac wird weiterverlinkt nach /usr/bin/javac

Zitat

The /etc/alternatives defines default applications for the traditional Unix/Linux system - such as, the default application to handle irc, to browse the web etc.


So, was heißt das?
Am besten du schaust dir dazu die Beispiele auf diese Seite an ( ist in engl. ) >klick<

Wenn du Java Versionen installiert hast, kannst du per /etc/alternatives festlegen, welche javac Version verwendet werden soll.
Genauso mit dem Beispiel von den Editoren auf der Seite die ich dir verlinkt habe.
